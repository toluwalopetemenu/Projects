# Data Science & Analytics Portfolio

Welcome to my Data Science and Analytics Portfolio.
This repository contains a curated collection of personal and technical projects demonstrating my skills in:
1. Data cleaning & preparation
2. Exploratory data analysis (EDA)
3. Machine learning (supervised & deep learning)
4. Data visualisation
5. Statistical reasoning
6. Reporting & insight generation

Each project is stored in its own folder with a dedicated README explaining the purpose, tools used, analysis steps, and key findings.

This portfolio has been created to showcase my practical experience working with:

4. Machine learning workflows
5. EDA and insight communication

The projects highlight my ability to work with real-world datasets, draw meaningful conclusions, and present results clearly.
